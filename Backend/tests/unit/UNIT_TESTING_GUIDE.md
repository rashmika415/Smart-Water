# Unit Testing

This section validates individual functions and controllers in isolation.

## Coverage Areas

- Controllers: `tests/unit/controllers`
- Services: `tests/unit/services`
- Utilities: `tests/unit/utils`

## Commands

From `Backend` folder:

```bash
npm run test:unit
npm run test:unit:saving-plan
npm run test:unit:usage
npm run test:unit:my-utils
npm run test:unit:my-controllers
```

## Purpose

Use unit tests to quickly verify business logic and edge cases without full module integration.
