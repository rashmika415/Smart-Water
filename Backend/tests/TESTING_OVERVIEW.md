# Backend Testing Guide

This folder contains testing split into separate parts:

- API smoke scripts: `tests/API`
- Unit tests: `tests/unit`
- Integration tests: `tests/integration`
- Performance tests: `tests/performance`

## Quick Start

From the `Backend` folder:

```bash
npm install
npm test
```

## Section Guides

- See `tests/API/API_SMOKE_TESTING.md` for API smoke scripts.
- See `tests/unit/UNIT_TESTING_GUIDE.md` for unit testing.
- See `tests/integration/INTEGRATION_TESTING_GUIDE.md` for integration testing.
- See `tests/performance/PERFORMANCE_TESTING_GUIDE.md` for load and spike testing.
